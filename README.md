# torrent-client

# Project Description

A Bittorrent client for peer to peer sharing of files.

Can download as well as seed files.

# Supported Platforms
- Ubuntu 18.04
- Windows

# Requirements
- random
- socket
- binascii
- logging
- urllib
- urllib.parse
- struct 
- select
- os
- hashlib
- bencodepy
- threading
